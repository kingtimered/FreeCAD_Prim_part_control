# 使用Python二次开发 FreeCAD  如何确定自己的编程环境
# How to determine your own programming environment while using Python to do FreeCAD secondary_development?
**结论先说清楚：你不需要自己选择 PyQt5 / PyQt6，也不需要单独决定 Python 版本。在 FreeCAD 里，这些都已经被内置并固定了，你只需要“跟着它用”。**

在安装之后需要确定环境，最好直接运行
在 FreeCAD 的 Python 控制台里执行：
```py
import sys
print(sys.version)
# 然后就是版本信息
3.11.14 | packaged by conda-forge | (main, Oct 13 2025, 14:00:26) [MSC v.1944 64 bit (AMD64)]
```

再执行：(Qt version) 观察版本

```py
from PySide import QtCore
print(QtCore.qVersion())
#结果
6.8.3
```
在测试一下是 Pyside2还是Pyside6
```py
import PySide2
print(PySide2.__version__)
```
```py
import PySide6
print(PySide2.__version__)
```
哪个没报错就是用哪个写的UI

UI还可以用命令确认
```py
import sys
print([m for m in sys.modules if "PySide" in m])
```
